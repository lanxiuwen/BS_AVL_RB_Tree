
# 二叉搜索树系列实现

适用泛型编程实现了**二叉搜索树**，**AVL平衡二叉树**及**红黑树类**

均支持以下操作

**insert** 插入
**remove** 删除
**successor** 查找某节点的后继节点
**predecessor** 查找某节点的前驱节点
**destory** 销毁元素

**遍历支持**
**preOrder** 前序遍历
**inOrer** 中序遍历
**postOrder** 后续遍历
**layerOrder** 层序遍历

**search** 查找
**search_min** 最小值查找
**search_max** 最大值查找

对于AVL树与红黑树支持计算输的高度
**height**

对于红黑树
**支持判断当前是否为合法红黑树**
**check**
